## 方式一：修改dockerfile文件

在dockerfile文件中添加如下指令：

```dockerfile
RUN sh -c 'touch /server.jar' && \
    apk add tzdata && \
    cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime && \
    echo Asia/Shanghai > /etc/timezone && \
    apk del tzdata
```

## 方式二：直接在容器中修改

执行以下shell：

```shell
sh -c 'touch /server.jar' && \
apk add tzdata && \
cp /usr/share/zoneinfo/Asia/Shanghai /etc/localtime && \
echo Asia/Shanghai > /etc/timezone && \
apk del tzdata
```
